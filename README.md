### Projects

- **[PSX Market Data Pipeline](https://github.com/karankumar24/fincom-psx-pipeline)**
  Grabs daily stock price data from the Pakistan Stock Exchange, handles a bunch of different file formats, fills in gaps for missing trading days, and puts it all together into one clean dataset. Written in Python with pandas and CI through GitHub Actions.

- **[LabOS](https://github.com/karankumar24/pdfProtocols)**
  You give it a research paper as a PDF and it generates step-by-step lab protocols, then checks them for safety, comprehension, and alignment with the source material. The backend is Python, the frontend is TypeScript, and the whole thing runs on Docker and Kubernetes with live progress updates over WebSockets.
